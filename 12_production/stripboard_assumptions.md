## Working Principle

The stripboard should be built around production containers, not around episode purity. Because `WE OUT HERE` is a hybrid show, the first pass must separate:

- recurring standing-set live action
- practical location days
- dedicated desktop realism capture
- surreal insert units

## Global Stripboard Assumptions

### Assumption Set A: Day Count

- Base live-action days: `24`
- Base desktop capture days: `4`
- Base surreal / pickup / plate days: `2`
- Hold or recovery days: `2`
- Total working shoot envelope: `30 + 2 hold`

### Assumption Set B: Page Load

- average full live-action day target: `4.5-5.0 pages`
- average standing-set dialogue day target: `5.5 pages max`
- average practical location day target: `3.0-4.0 pages`
- desktop capture day target: defined by screen sequence count, not page count
- surreal insert day target: defined by shot list completion, not page count

### Assumption Set C: Cast Load

- all three leads recur heavily enough to avoid strict episode-by-episode scheduling
- day-out-of-days should assume two-lead and three-lead days are the norm
- single-lead isolation material should be used as schedule ballast and recovery material

## Boarding Categories

### Category 1: Standing Set Material

Tag all scenes that can be shot in recurring interiors as first-priority board material.

Expected standing-set buckets:

- Cataclysm base operations
- personal workspace / domestic spill space
- hall / threshold / utility areas
- screen-facing webcam or desk scenes tied to those same spaces

### Category 2: Practical Location Material

Only board practical locations that materially expand story value. Each practical should justify itself by doing one of the following:

- show institutional pressure
- show external legitimacy or fraud risk
- show the trio trying to perform professionalism in public
- give scale or tonal contrast unavailable on a standing set

### Category 3: Desktop Realism Material

Board screen sequences as production assets with their own prep and approvals.

Each desktop board entry should eventually identify:

- episode and scene reference
- user perspective
- interface package version
- scripted beats
- whether live playback is needed on set
- whether capture is solo or synced with live-action performance

### Category 4: Surreal Insert Material

Board only after a sequence has:

- narrative purpose
- shot concept
- practical plan
- VFX burden estimate

If any one of those is missing, the sequence stays out of the first stripboard.

## Preliminary Production Blocks

| Block | Days | Material Bias | Notes |
| --- | ---: | --- | --- |
| Block A | 1-6 | Standing Set A material across all episodes | Fastest way to bank pages and establish performance rhythm |
| Block B | 7-10 | Practical location cluster 1 and 2 | Public / institutional material consolidated |
| Block C | 11-15 | Standing Set B and fallout interiors | Emotional and domestic scenes |
| Block D | 16-19 | Desktop realism capture block | Screens, calls, coverage support, evidence trail scenes |
| Block E | 20-24 | Remaining practicals plus selected standing-set pickups | Finish open loops before final stretch |
| Block F | 25-28 | Surreal inserts, finale material, pickups | Requires locked prep and strong post visibility |
| Block G | 29-30 | Buffer/live-action cleanup | Use only if needed; otherwise reduce spend |

## Scheduling Rules For First Board Pass

1. Never put heavy desktop capture on the same day as a major company move.
2. Never schedule surreal inserts as an afterthought at the tail of an overloaded day.
3. Keep first-week work inside controllable interiors unless cast or location constraints force otherwise.
4. Put high-risk practicals earlier enough that a loss can be recovered.
5. Treat evidence-bearing screen scenes as continuity-critical.
6. Keep one half-day pickup lane available each week where possible.

## What Is Deliberately Excluded From This Pass

- exact scene numbers
- exact page counts by scene
- background counts by scene
- final stunt or special equipment notes
- final meal break positions
- exact day-out-of-days

Those belong to later AD and line producing passes once scripts, cast, and locations are more stable.

## Red Flags For Reboard

- more than `8` distinct practical location addresses
- more than `2` unique UI builds per episode beyond the base chat/productivity stack
- more than `3` night exteriors for the season
- any episode rising above `5 total shoot-equivalent days`
- lead cast avail restrictions that break standing-set block logic

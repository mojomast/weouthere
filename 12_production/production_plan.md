## Production Mandate

- Title: `WE OUT HERE`
- Format: `6 x 30 min`
- Creative model: `desktop realism + live action + surreal inserts`
- Delivery model: contained indie professional production with controlled scope, paid crew, limited locations, and a repeatable post pipeline

This season should be produced like a smart contained series, not like a feature spread across six episodes. The plan prioritizes:

- recurring controllable locations
- block shooting across episodes
- a small but professional crew
- practical screen capture rather than inventing UI in post after the fact
- limited surreal work with clear approval gates
- schedule discipline over coverage bloat

## Base Production Model

### Planning Assumptions

- Target prep: `8 weeks`
- Target principal photography: `30 shoot days` plus `2 hold/buffer days`
- Target weekly pattern: `5-day weeks`, no planned weekends
- Target day length: `12 hours door to door max`
- Target company moves: no more than `1 major move` per day
- Target episodes per block: multiple episodes grouped by location and cast availability, not shot one episode at a time
- Target crew size on full live-action days: `32-40`
- Target crew size on desktop capture / insert days: `10-16`

### Shooting Day Types

| Day Type | Use Case | Crew Profile | Planning Rule |
| --- | --- | --- | --- |
| Full unit live-action | Dialogue scenes, ensemble scenes, recurring interiors | Full crew | Max 5 script pages unless highly controlled standing set work |
| Lean live-action | Small cast scenes, inserts, simple practicals | Reduced lighting/grip/art | Use to absorb schedule pressure without burning full-unit costs |
| Desktop capture | In-world screens, chat flows, browser action, recorded calls | Director, script supervisor, operator, UI/graphics, sound as needed, DIT/data support | Treat as shoot days, not post errands |
| Surreal insert | Stylized practical plates, performance fragments, VFX plates | Small specialty unit | Pre-board and pre-approve every beat |
| Splinter unit | Pickups, MOS inserts, environment, props, transitions | Minimal crew | Only with AD and line producer approval |

## Scale Strategy

### What We Spend On

- lead cast performance time
- controllable standing environments
- strong production sound
- camera and data discipline
- practical screen pipeline
- art/graphics integration that prevents expensive post fixes

### What We Do Not Spend On

- constant location churn
- unnecessary night work
- speculative VFX-heavy sequences without locked boards
- oversized background counts
- coverage patterns better suited to prestige feature budgets

## Season Execution Strategy

### Block Shooting Priorities

1. Shoot standing-set material across episodes first.
2. Consolidate each practical location into as few visits as possible.
3. Isolate desktop realism scenes into dedicated capture days or partial days.
4. Contain surreal inserts to preselected windows so camera, art, and post can prep properly.
5. Protect cast-heavy days from unnecessary company moves.

### Episode Weighting

- `S01E01` and `S01E02`: heaviest on setup, chat/UI, and legitimacy-building spaces
- `S01E03` and `S01E04`: heaviest on escalation, performance tension, and controlled surreal drift
- `S01E05` and `S01E06`: heaviest on consequence, exposure, and payoff material

The budget and schedule should assume uneven load across episodes, but even load across the whole season. Hard episodes are balanced by lighter contained episodes.

## Recommended Standing Environments

### Standing Set A: Cataclysm Base

- primary group work environment
- modular enough to play startup office, apartment work zone, and all-night war room
- supports multiple screen setups and practical motivated lighting
- highest return on recurring use

### Standing Set B: Personal Spill Space

- redressable domestic environment for one-on-one or isolated scenes
- can cover more than one character context with lighting and layout changes
- supports emotional fallout scenes and late-season depletion

### Standing Set C: Hallway / Threshold / Utility Zone

- low-cost connector space
- useful for transitions, pickups, phone calls, and tension scenes
- can absorb schedule losses when a larger setup drops out

## Core Department Priorities

### AD / Line Producer

- enforce day-type logic and company move limits
- carry one interior cover set option each week
- flag script pages that require screen builds at least `7 days` before shoot
- schedule surreal work only after boards, tech notes, and post sign-off exist

### Art / Graphics

- build the world as a near-present, overextended tech workspace, not glossy sci-fi
- treat all in-frame graphics as story-critical props
- lock hero graphics before camera tests

### Camera / DIT / Post

- maintain one camera language for live action and one deliberate grammar for desktop realism
- preserve metadata linking screen captures to scene and take
- keep capture formats editorially simple and consistent

### Sound

- prioritize clean dialogue capture over aggressive noisy locations
- record device playback, notification references, and room tone systematically
- capture wild lines on the day when interface-driven dialogue timing matters

## Contingency Rules

### Automatic Escalation Triggers

- any episode forecast above `5.5 live-action days`
- any location requiring more than `2 company moves` to cover material already planned elsewhere
- any new script beat needing custom UI not already scoped
- any surreal sequence without boards, plate list, and post estimate
- any location sound issue that threatens dialogue-heavy material

### Primary Contingency Moves

1. Move scenes into standing environments where story allows.
2. Convert low-value exterior beats into interior or threshold beats.
3. Collapse coverage into stronger singles / two-shots where performance supports it.
4. Shift desktop material into dedicated capture day rather than forcing it into an overloaded live-action day.
5. Cut or simplify surreal beats before cutting cast-facing material.

## Success Criteria

- season completes in `30 principal days` or fewer, excluding optional hold days
- no department invents a separate screen pipeline
- standing sets absorb at least `50%` of live-action page count
- practical locations are limited enough to permit block shooting
- post receives organized media, metadata, and graphics source packages from day one
- surreal work remains additive and specific, not a rescue plan for unclear scenes

## Table Read Summary
The episode reads as a clean tonal pivot from private contamination to public consequence. The screenshot plus rogue scheduled post is a strong one-two opening. `Owen Kaye` is especially effective because he does not overperform judgment. The key table-read opportunity is triangulation: the scandal matters most when the audience can track the difference between optics failure, pattern failure, and authorship resentment all at once.
